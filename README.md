# 3D-Code

##  アプリ概要
- 色情報と文字情報を組み合わせた３次元コード
- ２次元コードよりも情報の種類を増やしたため、情報格納量が増加  
- 誤り訂正は、QRコードにも採用されているリードソロモン符号

## 使用技術
### Server
- Docker (APIサーバー)
- ngnix (Webサーバー)

### Python3.11.7
- openCV (画像処理)
- PyOCR (文字認識)
- Reedsolo (誤り訂正)
